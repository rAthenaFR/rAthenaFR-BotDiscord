# Code de conduite

## Objectif

rAthenaFR Discord Bot est un projet open source orienté communauté. Les échanges doivent rester techniques, respectueux et centrés sur la sécurité, la maintenabilité et la lecture seule de la base de données.

## Comportement attendu

Les contributeurs et mainteneurs doivent :

- rester respectueux et constructifs ;
- concentrer les retours sur le code, la documentation et le comportement du bot ;
- accepter qu’une contribution puisse être refusée si elle affaiblit la sécurité ou la confidentialité ;
- ne jamais partager de tokens, mots de passe, données de comptes ou informations serveur privées ;
- aider les contributeurs ayant différents niveaux d’expérience.

## Comportement interdit

Sont interdits :

- harcèlement, menaces, insultes ou attaques personnelles ;
- langage discriminatoire ;
- publication d’informations privées sans accord ;
- partage de vrais identifiants, tokens, mots de passe, emails, PINs, IPs ou données de compte ;
- contournement volontaire des règles de sécurité du projet.

## Signalement

Contacte les mainteneurs par un canal privé si un signalement contient des informations sensibles. Ne poste jamais de preuve sensible dans une issue publique.

## Application

Les mainteneurs peuvent modifier, fermer ou verrouiller des discussions, refuser une contribution ou restreindre une participation si nécessaire pour protéger le projet Athena.
